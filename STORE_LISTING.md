# Chrome Web Store Listing

## Extension Name
Anime Filler Checker — Skip the Filler!

## Short Description (132 chars max)
Auto-detects anime and episode on streaming sites, shows a FILLER or CANON badge directly on the page.

## Detailed Description
Never waste time on filler episodes again!

Anime Filler Checker automatically detects the anime name and episode number from the streaming site you're watching, cross-references it with AnimeFillerList.com, and instantly shows a floating badge on the page:

✅ CANON — Follows the manga, safe to watch
⛔ FILLER — Not from the manga, safe to skip
⚠️ MIXED — Mix of canon and filler content
🔵 ANIME CANON — Anime-original but part of the story

★ KEY FEATURES:
• Auto-detection of anime name and episode from the page
• On-page floating badge — see the verdict without opening the popup
• Manual lookup — type any anime name and episode number
• Smart fuzzy search — finds anime even with imperfect names
• MyAnimeList score shown on badge and popup
• Watch history — track your recently checked episodes
• Streaming site whitelist — control which sites activate the badge
• Badge customization — position, content, and auto-hide timer
• Draggable badge — move it anywhere on the page
• Fast local cache — instant results on repeat visits

★ HOW IT WORKS:
1. Navigate to any anime episode on a streaming site
2. The extension detects the anime and episode automatically
3. A floating badge appears showing the filler status
4. Click the extension icon for manual lookup or details

Data sourced from AnimeFillerList.com.
Anime scores provided by MyAnimeList via the Jikan API.

Open source and collects no personal data.
